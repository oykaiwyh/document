```css
        separate	默认值。边框会被分开。不会忽略 border-spacing 和 empty-cells 属性。
        collapse	如果可能，边框会合并为一个单一的边框。会忽略 border-spacing 和 empty-cells 属性。
        inherit	规定应该从父元素继承 border-collapse 属性的值。

  				  border-spacing:0;
		border-collapse 被设置为 separate否则将忽略这个属性

2.圆角的设定
	在 w3c 上有介绍说，border-radius 属性是一个简写属性，用于设置四个 border-*-radius 属性 ，分别是以下组成：
    左上圆角半径：border-top-left-radius，
    右上圆角半径：border-top-right-radius，
    右下圆角半径：border-bottom-right-radius，
    左下圆角半径：border-bottom-left-radius

									clip:修剪文本
									string:使用给定的字符串来代表被修剪的文本(火狐)
4.解决span img等排版不在同一条直线上
	设置图片属性：
    img{
        vertical-align:bottom;//两者居中对齐
    }

5.弹性布局
	前提：	display:flex;//前提 父元素设置弹性布局才能设置弹性属性
	属性：
		盒子水平属性：justify-content:space-between/flex-end....; 
		盒子垂直属性：align-items:center/top...; 

6.内联块没有对齐--两个节点一高一低
	问题：当块级元素包含内联元素时，内联元素默认是和块元素的基线baseline对齐的，比如例子中的文字“Girl”和图片img均默认的和div的基线对齐，而基线距离块元素边框默认的情况下是具有一定的距离的，从而会产生间距。margin和padding都无法解决该问题。可能回出现一高一低
	解决：vertical-align：top/middle  //把元素的顶端与行中最高元素的顶端对齐/在父元素的中部。
	内联元素display: block

7. 设置元素的默认尺寸 initial
	列表：list-style:none/initial  //无/默认/列表的样式
	width/height:initial //默认的样式

8.span等等标签超出宽度自动换行
    span{
        word-break:normal; 
        width:auto; 
        display:block; 
        white-space:pre-wrap;
        word-wrap : break-word ;
        overflow: hidden ;
    }
    white-space -- 通过HTML文档的源代码的排版方式控制页面显示文本的排版方式 
    取值: normal | pre | nowrap | pre-wrap | pre-line | inherit 
    normal: 正常无变化(默认处理方式.文本自动处理换行.假如抵达容器边界内容会转到下一行) 
    pre: 保持HTML源代码的空格与换行,等同与pre标签 
    nowrap: 强制文本在一行,除非遇到br换行标签 
    pre-wrap: 同pre属性,但是遇到超出容器范围的时候会自动换行 
    pre-line: 同pre属性,但是遇到连续空格会被看作一个空格 
    inherit: 继承 
    初始值: normal 
    继承性: 是 
    适用于: 所有元素 
    white:白色.space:间隔,距离
```

```js
1.mouseover/mouseout事件，显示内容出现闪烁情况
	描述：鼠标覆盖显示之后如过显示内容出现在鼠标下面，则会出现显示内容闪烁
    原因：显示内容在鼠标之下，覆盖了定义mouseover的dom，mouseout时间事件则会触发，导致一直循环，最终出现内容闪烁。
    解决：增加属性 pointer-events: none   //ie11+
    
2.平分字符串
	function splitChunks(string，blocknum) {
	//strting：平分字符串
	//blocknum： 平分为几块
	var regex = RegExp(".{1,"+Math.ceil(string.length/blocknum)+"}", 'g');
    return string.match(regex);
}

3.数组查找某个字符，

	array.contains('key') //

4.检查数组每个值是否相同
	every() 方法用于检测数组所有元素是否都符合指定条件（通过函数提供）。
    例：判断数组每一个值是否相等
    var array = ['success','success','success','success1'];
    var a1 = 'success';
    console.log(array.every(item=>item===a1)) //false

5.解决异步问题
解决方案：
把var改为let，let是块极作用域，每次for循环都会把对应的i绑定到添加的任务【闭包函数，setTimeout()中的函数】中,所以当主线程执行完时，也不会影响到每个任务中i。所以可以打印出0   1   2
    把定时器放在一个自执行函数中用i当做参数
    (function(i){
          setTimeout(function(){
                console.log(i);
           },1000)
    })(i)
```
